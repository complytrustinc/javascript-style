# Stripe JavaScript Linting Rules

### Usage

Add `stripe-javascript-style` to your `devDependencies`.

```bash
> npm install --save-dev stripe-javascript-style
```

In `.jscsrc`:

```json
{
  "plugins": ["stripe-javascript-style"],
  "preset": "stripe"
}
```
